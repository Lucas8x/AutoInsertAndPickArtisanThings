</p>

## Requirements
- [SMAPI 4.x](https://smapi.io/)
- [Generic Mod Config Menu](https://www.nexusmods.com/stardewvalley/mods/5098)


## Usage
Hold/Equip any fruit/vegetable/ingredient/product to automatically put that item in the nearby machines.

- **Auto P√¨ck**: Automatically pick up ready items from the machine

- **Insert Keybind**: Keybind to manually insert ingredients on machines within the radius. Remember to disable auto insert.
- **Pickup Keybind**: Keybind to manually pick up ready items from the machines within the radius. Remember to disable auto pickup.
